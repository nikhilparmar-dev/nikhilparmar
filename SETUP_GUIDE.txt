Step 1 — Install Required Libraries
-------------------------------------
Open terminal/command prompt aur ye run karo:

    pip install flask supabase python-dotenv werkzeug openpyxl reportlab gunicorn


Step 2 — Supabase Setup
-------------------------------------
1. supabase.com pe jaao
2. New account banao (free)
3. "New Project" click karo
4. Project name: rajivnagar-shala
5. Password set karo (yaad rakhna)
6. Region: Southeast Asia select karo
7. Create Project click karo

TABLES BANAO (SQL Editor mein ye paste karo):

-- users table
CREATE TABLE users (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    name TEXT NOT NULL,
    username TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    role TEXT NOT NULL CHECK (role IN ('teacher', 'principal')),
    class_assigned TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);

-- students table
CREATE TABLE students (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    name TEXT NOT NULL,
    roll_no TEXT NOT NULL,
    gr_number TEXT,
    class TEXT NOT NULL,
    dob TEXT,
    gender TEXT,
    caste TEXT,
    parent_contact TEXT,
    aadhaar_number TEXT,
    bank_account TEXT,
    aadhaar_doc_url TEXT,
    ration_doc_url TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);

-- attendance table
CREATE TABLE attendance (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    student_id UUID REFERENCES students(id) ON DELETE CASCADE,
    date DATE NOT NULL,
    status TEXT NOT NULL CHECK (status IN ('P', 'A', 'L')),
    marked_by UUID,
    created_at TIMESTAMP DEFAULT NOW()
);

-- marks table
CREATE TABLE marks (
    id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
    student_id UUID REFERENCES students(id) ON DELETE CASCADE,
    subject TEXT NOT NULL,
    semester TEXT NOT NULL,
    written_marks INTEGER DEFAULT 0,
    participation_marks INTEGER DEFAULT 0,
    academic_year TEXT DEFAULT '2025-26',
    entered_by UUID,
    created_at TIMESTAMP DEFAULT NOW()
);

RLS DISABLE KARO (for simplicity):
    - Supabase dashboard > Authentication > Policies
    - Har table pe "Enable RLS" band karo
    OR ye SQL run karo:
        ALTER TABLE users DISABLE ROW LEVEL SECURITY;
        ALTER TABLE students DISABLE ROW LEVEL SECURITY;
        ALTER TABLE attendance DISABLE ROW LEVEL SECURITY;
        ALTER TABLE marks DISABLE ROW LEVEL SECURITY;

STORAGE BANAO:
    - Supabase > Storage > New Bucket
    - Name: school-docs
    - Public: YES (toggle on)


Step 3 — .env File Banao
-------------------------------------
1. .env.example file copy karo
2. Rename karo -> .env
3. Supabase se values copy karo:
    - Project > Settings > API
    - "Project URL" copy karo -> SUPABASE_URL
    - "anon public" key copy karo -> SUPABASE_KEY
4. .env file mein paste karo


Step 4 — Principal Account Banao
-------------------------------------
Python terminal mein ye run karo ek baar:

    python3 create_admin.py

(create_admin.py file saath mein di gayi hai)


Step 5 — App Locally Run Karo
-------------------------------------
    python app.py

Browser mein jaao: http://localhost:5000


Step 6 — Render.com pe Deploy Karo
-------------------------------------
1. GitHub pe repository banao
2. Sab files push karo
3. render.com pe jaao - New Web Service
4. GitHub repo connect karo
5. Environment Variables add karo:
    - SUPABASE_URL = (apna value)
    - SUPABASE_KEY = (apna value)
    - SECRET_KEY = rajivnagar-shala-2025
6. Deploy!
7. Live link milega - school ko de do
